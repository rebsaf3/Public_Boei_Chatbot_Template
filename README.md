# Public Boei Chatbot Template
Public website assistant setup package

Last updated February 08, 2026

## What this repo contains
This repository is a template for setting up a public website chatbot in Boei with strong guardrails.
All organization names, phone numbers, and specific identifiers have been removed.
Replace placeholders with client approved values before use.

## Design goals
1 No hallucination and no guessing
2 Approved sources only
3 No escalation promises
4 No contact capture
5 Short direct answers and clear routing for crisis content

## Platform constraints
1 Specific Instructions have a 500 character limit each
2 FAQ builder supports up to 10 items per prompt
3 Website training may be limited by trained page caps

## Repo map
1 docs includes the SOP and Quick Start
2 configuration includes copy ready text for the Boei setup
3 prompts includes standard tests and FAQ batch guidance
